# Rainloop Docker

# Usage
```
docker run -d --name rainloop -p 8085:80 martadinata666/rainloop
```

## Tweak as you need
http://ip-address:8085/?admin

Here you can add domain mailbox, change admain password, etc.

## Default admin login
```
User: admin
Pass: 12345
```
